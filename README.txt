Roadie Support Training Simulator — Proof of Concept

Open index.html in a modern browser. No installation, server or internet connection is required.

Scenario flow:
1. Enter Practice Mode.
2. Start “Resolve duplicate driver profiles.”
3. Accept Avery Morgan’s incoming chat.
4. Use the phone number from the chat to search Admin.
5. Review at least two related profiles.
6. Mark TRN-ACC-001 as primary and unlock it.
7. Clear license information from TRN-ACC-002 and TRN-ACC-003.
8. Add notes to the primary and both duplicate profiles. Notes should identify the duplicate relationship and primary profile.
9. Return to the chat and send a response.
10. Submit the scenario from the Training Coach panel.

All names, IDs, account data and actions are fictional.
